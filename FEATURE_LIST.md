# ChickenNoteLM Feature List

> 更新时间：2026-03-15  
> 部署路径：`/var/www/chickennoteLM`

## 1) 账号与访问控制

- 用户登录（用户名登录，服务端 Session）
- 登录态校验：页面初始化时调用 `/api/auth/me`
- 未登录强制停留在登录页（登录遮罩）
- 退出登录（`/api/auth/logout`）
- 按用户隔离数据目录（`notefile/<user_id>`、`chatdata/<user_id>`、`eventdata/<user_id>`、`sync_state/<user_id>`）

## 2) 笔记编辑与管理

- 新建笔记
- 笔记列表展示（标题、更新时间、内容摘要）
- 切换笔记加载正文
- 自动保存（输入防抖）
- 手动保存按钮
- 重命名当前笔记
- 笔记多选、全选、批量删除
- Markdown 编辑/预览模式切换

## 3) 数据存储与同步

- 本地存储：
  - IndexedDB（主离线存储）
  - localStorage（兼容与设置存储）
- 服务端存储：
  - 笔记落盘（`*.md + *.json`）
  - 事件落盘（`events.json`）
- 同步能力：
  - 全量同步接口：`/api/sync/notes-events`
  - 增量拉取：`/api/sync/pull`
  - 增量推送：`/api/sync/push`
  - 同步状态查询：`/api/sync/status`
  - 冲突解决接口：`/api/sync/resolve`
- 网络状态感知（online/offline）与后台同步触发

## 4) AI 能力

- AI 对话（基于当前选中笔记上下文）
- 聊天会话管理（新建、读取、保存、列表）
- 事件提取（从笔记中抽取结构化事件）
- 文本转 Markdown（按钮触发一次模型调用）
- AI 转换结果一键回写编辑器并自动保存
- AI 转换历史支持撤销（最近记录）

## 5) 可配置项（Settings）

- API Key
- Base URL
- 模型名称
- 对话系统提示词
- 事件提取系统提示词
- 文本转 Markdown 提示词（可自定义）
- 设置本地持久化（`chickennotelm_settings`）
- 旧配置键兼容迁移（历史字段自动兼容）

## 6) 事件模块

- 单篇提取事件、全局提取事件
- 事件卡片列表展示
- 事件详情展开/折叠
- 事件标题/上下文编辑
- 标签新增、删除、按标签过滤
- 事件排序（创建时间、更新时间，升降序）

## 7) 导入导出

- 导入 JSON 数据（笔记/事件）
- 导出笔记 JSON
- 导出事件 JSON
- 导出所有笔记为单个 Markdown 文件

## 8) 前端离线与缓存

- Service Worker 注册
- 静态资源缓存
- API 请求网络优先
- 缓存版本管理（当前缓存名：`chickennoteLM-v4`）

## 9) 后端能力（Flask）

- 笔记 API：`/api/notes`
- 事件 API：`/api/events`
- 聊天 API：`/api/chats`、`/api/chats/<id>`
- 鉴权 API：`/api/auth/login`、`/api/auth/me`、`/api/auth/logout`
- 同步 API：`/api/sync/*`
- OpenClaw 代理接口：
  - `/api/openclaw/health`
  - `/api/openclaw/chat`
  - `/api/openclaw/tui`（WebSocket）

## 10) 已知约束（供开发参考）

- 无痕模式下浏览器本地缓存（IndexedDB/localStorage/cookie）不稳定，可能导致会话结束后数据不可见
- 模型能力依赖外部 API 可用性与 Key 配置
- 当前登录是轻量用户名模式，适合内测，不等同于生产级认证体系
